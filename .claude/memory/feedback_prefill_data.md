---
name: Always pre-fill with available data
description: Never give empty templates when you already have the data to fill them. Pre-fill everything, only leave genuinely unknown fields blank.
type: feedback
---

When creating questionnaires, templates, or any document that collects information — if the user has ALREADY given you the data (transcripts, notes, images, prior sessions), PRE-FILL everything with that data. Don't hand back empty fields asking for information you already have.

**Why:** User gave 26 handwritten pages, a CTO PDF, a voice transcript, and multiple sessions of context. I created an empty questionnaire asking him to fill in answers he'd already given me. This wasted his time and showed I wasn't using the data he provided.

**How to apply:**
- Before creating any template/questionnaire, check: do I already have this information?
- If yes, pre-fill it. Only leave blank what's genuinely unknown.
- The user's time is the most valuable thing. Never make them repeat themselves.
- "I have the data but gave you empty fields" is unacceptable.
