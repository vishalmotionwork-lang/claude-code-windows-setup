# Claude Code setup (Windows)

Installed by the one-command bootstrap. Layout:
- CLAUDE.md         global operating rules (loaded every session)
- settings.json     permissions + plugin marketplaces + startup hook
- rules/            coding standards (common + per language)
- memory/           MEMORY.md index, USER.md, feedback_*.md rules, reference_*.md, projects/

Credentials are intentionally NOT here. Add per-project creds locally; never commit them.
Rebuild your machine anytime with the same one-liner.
