# Content Creation

## Source Citations (Mandatory)

Every content output MUST include source references:

- YouTube: `[Source: "Video Title" @ MM:SS]`
- PDF: `[Source: "document.pdf" p.N]`
- Instagram: `[Source: @handle, post YYYY-MM-DD]`
- Local file: `[Source: "filename.ext"]`
- Multiple: `[Source: "Video A" @ 2:15; "doc.pdf" p.3]`

Never produce content analysis or derivative content without citing the specific source and location.

## Content Analysis Checklist

When analyzing any content source, always cover:

1. **Hook**: Type (question/claim/story/stat/interrupt), effectiveness, curiosity gap
2. **Structure**: Framework, transitions, pacing, section breakdown
3. **CTA**: Primary ask, placement, soft CTAs, urgency techniques
4. **Engagement**: Retention hooks, pattern interrupts, social proof, emotional arc

## Output Format Rules

- Scripts: Include timing markers and delivery notes
- Posts: Hook line first, 3-5 short paragraphs, CTA at end
- Analysis: Use structured format with Overview, Hook, Structure, CTA, Takeaways sections
- All formats: End with `Sources:` section listing all references

## Knowledge Base Hygiene

- Save loaded sources to `~/.claude/projects/-Users-vishal-motion/memory/sources/`
- Use naming convention: `yt-<id>`, `pdf-<slug>`, `ig-<id>`, `src-<slug>`
- Update INDEX.md when adding/removing sources
